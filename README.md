
##项目运行
```
git clone https://github.com/xiaonizi66/vue-ssr-demo  
```
## Project setup
```
npm install
```
## build sever
```
npm run server:build
```
## build cilent
```
npm run cilent:build
```
## run server
```
nodemon server.js
```
